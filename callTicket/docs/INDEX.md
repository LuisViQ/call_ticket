# Indice de Documentacao

Esta pasta documenta o codigo do app por secao. Cada arquivo na raiz do projeto e em src/ esta coberto em um dos documentos abaixo.

-> ROOT.md: arquivos da raiz e assets estaticos
-> ROUTES.md: stacks de navegacao e troca de rotas
-> CONTEXTS.md: contexto de auth e bootstrap da sessao
-> SERVICES.md: funcoes e tipos do cliente da API
-> UTILS.md: helpers do AsyncStorage
-> COMPONENTS.md: componentes de UI reutilizaveis
-> SCREENS.md: todas as telas e estilos
